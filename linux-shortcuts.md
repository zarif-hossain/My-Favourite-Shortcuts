# Mint Cinnamon Shortcuts

*note: Super key is Windows key*

## Workspace Management

**Ctrl+Alt+Up-Arrow**
:   Show all workspaces

**Ctrl+Alt+Left-Arrow**
:   Go back in workspace

**Ctrl+Alt+Right-Arrow**
:   Go forward in workspace

**Ctrl+Shift+Alt+Left-Arrow**
:   Move window to Left

**Ctrl+Shift+Alt+Right-Arrow**
:   Move window to Right

**Super+Shift+Right-Arrow**
:   Move window to Right Monitor

**Super+Shift+Left-Arrow**
:   Move window to Left Monitor

## Windows

**Ctrl+Alt+Down-Arrow**
:   Show open windows in current workspace

**Alt+Tab**
:   Cycle windows

**Super+d**
:   Show desktop

**Alt+space**
:   Activate window menu

## Windows Tiling

**Super+Left**
:   Push tile left

**Super+Right**
:   Push tile right

**Super+Up**
:   Push tile Up

**Super+Down**
:   Push tile Down

## System

**Ctrl+Alt+l**
:   Lock screen

**Super+e**
:   Opens Home file

## Terminal
**Ctrl+l**
:   Clear terminal

## Launchers

**Ctrl-Alt+t**
:   Open Terminal
